# [Brittany Tuwamo] - Data Analysis Portfolio

Welcome! This repository contains the complete template for a professional, one-page data analysis portfolio website, hosted for free on GitHub Pages.

## About This Portfolio Template

This portfolio is designed to demonstrate skills in:
- Data visualization and storytelling
- Data modeling and analysis
- Transforming complex data into actionable insights

## 🚀 Projects

### The Relationship of Digital Behavior and Mental Health
*A multiple linear regression analysis with SAS to evaluate the impact of digital behaviors influence mental wellbeing of students*

### EPA Air Pollution Concentrations
*A time series analysis of the levels of toxic pollution compounds from 2018 to 2020 and the contributing factors to pollution changes with high and low population density in southern states.*

## 🛠️ Skills

- **Data Visualization**: Power BI
- **Data Analysis**: SQL, Python (Pandas), R, Excel, SAS
- **Tools**: Power BI Service
